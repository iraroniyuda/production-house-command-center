# Product Principles

Status: Approved  
Owner: Roni / Product Owner  
Last Reviewed: 2026-07-22  
Applies To: Product & UX  
Related Documents:
- Belum ditentukan


1. Mulai dari masalah lapangan, bukan fitur keren.
2. Setiap fitur harus dapat dijelaskan dalam bahasa orang produksi.
3. Jika kru enggan mengisi data, alur harus disederhanakan.
4. UI boleh indah, tetapi harus cepat saat dipakai di lokasi shooting.
5. Sistem boleh besar, tetapi tiap user hanya melihat bagian yang relevan.
6. Sekali data masuk, data SHOULD mengalir ke schedule, call sheet, report, budget, dan dashboard.
7. Perubahan di satu tahap MUST menunjukkan dampaknya ke tahap lain.
8. Informasi kritis MUST dapat dibagikan melalui link yang aman dan mudah dibuka.
9. Dashboard MUST berasal dari data operasional nyata, bukan grafik dekoratif.
10. Production memory harus tumbuh dari penggunaan sistem, bukan dari input administratif tambahan.
