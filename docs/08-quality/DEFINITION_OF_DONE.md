# Definition of Done

Status: Approved  
Owner: Roni / Product & Engineering  
Last Reviewed: 2026-07-22  
Applies To: Feature Delivery  
Related Documents:
- Belum ditentukan


Feature selesai bila:

- acceptance criteria terpenuhi;
- business rule memiliki test;
- authorization diuji;
- tenant isolation diuji bila relevan;
- API/OpenAPI diperbarui;
- migration tersedia dan tervalidasi;
- observability ditambahkan untuk proses kritis;
- audit requirement dipenuhi;
- error handling sesuai catalog;
- dokumentasi feature diperbarui;
- Current Implementation diperbarui;
- Known Gaps diperbarui;
- CI lulus;
- tidak ada secret atau debug artifact.
