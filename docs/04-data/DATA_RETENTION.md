## Policy Awal

- Audit event: retain selama project aktif + periode arsip yang ditentukan.
- Published call sheet revision: retain selama project dan archive.
- Daily report: retain sebagai production record.
- Raw confirmation token: tidak pernah disimpan.
- Expired token hash: dapat dibersihkan setelah retention window.
- Temporary upload: dibersihkan otomatis.
- Failed notification payload: retain terbatas dan redacted.
- Logs: tidak boleh menjadi arsip bisnis permanen.
- Personal data: harus dapat dianonimkan atau dihapus sesuai kontrak dan kewajiban legal.
